# SEA: Add Assets to a Network Device

>**Note**: You can add assets directly from the [Quick Wizard](../secure_equipment_access/sea_quick_wizard.md) screen. Follow the steps covered in the **Connect to Asset** option.

**To manually add an Asset for a device**:

>**Note**: These assets can be subtended devices or the network device itself.

1. From the **System Management** screen, choose a **Network Device**.
2. From the **Network Device details** screen, click **Add Asset**.
3. After you choose or add an asset, click **Add**. The asset is associated with the network device.
